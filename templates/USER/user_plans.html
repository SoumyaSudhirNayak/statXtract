Frequently Asked Questions

Can I cancel my subscription at any time? +
Yes! You can downgrade your plan directly from the Dashboard Settings anytime. You will retain Pro access until your current billing cycle expires.
What counts as an API request for the Professional plan? +
Any programmatic GET request made to `api.statxtract.in/*` using your private integration token will count towards the 100 requests/hour limit. Manual queries through the Web Dashboard do not consume this quota.
Is the data up to date? +
We directly ingest datasets securely via official APIs and Government pipelines exactly as they update. However, data frequency relies wholly on the individual Indian dataset publish schedules.
